Power Adjacency探究

定义

一个图G的邻接矩阵记为A，我们将A的任意幂次记为A_k，A_k的元素a_{ij}表示从节点i到节点j的k跳路径数，我们称A_k为k阶power adjacency matrix。

性质

性质一

k1 > k2，则A_{k1} >= A_{k2}，即k阶power adjacency matrix的元素是单调递增的。
这就说明，A_{k1}中任意元素不为0代表的就是ij两点有小于等于k1跳的途径，反过来讲就是说A_{k2}不为零的元素在A_{k1}中一定不为零。

OmniParser 使用体验

介绍

OmniParser是微软推出的一个声称可以自动化控制电脑的项目

思路

用户上传图像 → 预处理 → 图标检测(YOLO) + 文本识别(OCR) → 图标语义理解(多模态模型) → 结构化输出
最终得到了图片上的各个元素以及他的坐标

使用方法

直接运行omnitool下的任意一个即可

OmniParser 使用体验

介绍

OmniParser是微软推出的一个声称可以自动化控制电脑的项目

思路

用户上传图像 → 预处理 → 图标检测(YOLO) + 文本识别(OCR) → 图标语义理解(多模态模型) → 结构化输出
最终得到了图片上的各个元素以及他的坐标

使用方法

直接运行omnitool下的任意一个即可
